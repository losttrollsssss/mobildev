# mobildev
Даны фигуры: круг, квадрат и прямоугольник. Из интерфейса Transforming реализованы функции resize и rotate. 
Из Movable функции Move для любых фигур.
На прямоугольнике с точками (0, 0), (6, 8) move(1, 1) в результате выдаст (1, 1), (7, 9)
Функция resize(4) на квадрате со стороной 2, сделает стороны квадарата равными 8
Функция rotate(Clockwise, 0, 0 на прямоугольнике с точкой (0, 3), шириной 2 и высотой 4, выдаст точку (3,0) и ширину с высотой равними 4 и 2
